***My Environment***
- Ubuntu 14.04
- Python 2.7
- Django 1.7.3

***Installing Django***
1. installing pip
	by downloading `get-pip.py` file and then running `sudo python get-pip.py`
2. installing Django
	by running `sudo pip install django`
3. creating first app
	by running `django-admin.py startproject votoyo`
